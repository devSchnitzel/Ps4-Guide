    Installing Ps2 Backups

    For support, ask for help at the Discord Server.
    If you appreciate this guide, I accept Donations.

    Table of Contents

    • Required Reading
    • What You Need

    Required Reading

    This is for installing the dumps of discs you have made for convienence, please do not use this guide for piracy!, click here for a guide to dump you games.

    What You Need

    • Ps4 with HEN or Mira CFW
    • The latest release of ps2 emu
    • The latest release of free ps2 pub gen
    • The latest release of Fake PKG Tools
    • Game backup you want to install

    Instructions

    1. Extract both ps2emu.rar and Fake PKG Tools
    2. Put your favorite PlayStation 2 game ISO in the ps2emu\image\ folder
    3. Rename the iso to 'disc01.iso'
      • the emulator can take multiple discs, but this is more advanced and you have to specify things in lua or the command line arguments
    4. Go to sce_sys and edit 'param.sfo' with 'orbis-pub-sfo.exe'
      • Change content ID to (UP9000-CUSAXXXXX_00-SCUS974900000001) where XXXXX is any string of numbers
      • The content ID can be somewhat random, but has to be in the enforced format. It doesn't really matter what you do, but the original emulated game (Rogue Galaxy) was like this and I just decided to change the title ID portion
      • Go to Title Text tab and change the text to the title of your game (ex. Ratchet & Clank, Crash Bandicoot)
    5. Save the 'param.sfo'
    6. In Fake PKG Tools, rename 'orbis-pub-cmd-ps2.exe' to 'orbis-pub-cmd.exe' (delete or rename original 'orbis-pub-cmd.exe')
    7. Open 'orbis-pub-gen.exe'
    8. Double click on 'Image0'
      • We are going to add the files from ps2emu, including 'disc01.iso'/'disc0X.iso'
    9. Drag and drop all the files from ps2emu into the Image0 root
    10. Go to the chunck tab, and assign all files to Chunck #0
    11. Exit image settings view
    12. Go to Command -> Project Settings -> Package
      • Paste in your content ID that you put in your 'param.sfo'
      • Generate a passcode, or use 'IvBkwybGjpTdC1uijHBWbQR-XSFXgKqc'
      • Change application type (at the bottom) to Freemium App
    13. Click build, select output directory, build
    14. If you get error "Could not create system file. sce_discmap.plt"
      • Hit close, double click Image0, try to reasign chunks (you should have done this already)
      • Then, try to build again. This seems to be some sort of glitch/bug?

    Section II - Installing Games

    1. Open "Settings"
    2. Scroll down to "Debug Settings", and press (X) to open it
    3. Scroll down and press (X) to open the "Game" menu
    4. Scroll down and press (X) to open the "Package Installer"
    5. Select the game to install
      • If you selected several games, select "Install All"
    6. After finishing the installation, press "(PS)" to return to the home screen
    7. The game(s) should now be on the homescreen, now, enjoy playing those classics
