    Required Reading

    This method of updating was introduced in 1.05, and it lets you install an update file directly.

    What You Need

    • PlayStation4 system
    • Computer/Laptop connected to the internet.
    • A USB memory device with at least 460MB of space.
      • Before starting, make sure the USB device is wiped clear.

      Instructions

        1. Using a computer, create a folder named 'PS4'. Inside that folder, create another folder named 'UPDATE'.
        2. Download the update file, and save it in the 'UPDATE' folder you created in step 1.
        • Save the file with the file name 'PS4UPDATE.PUP'.
        • [Download 5.05 here] Click to start the download.
        3. Now move the whole 'PS4' folder to the root of your USB storage device.
        • Connect the USB storage device on to your Ps4 system, and then from the function screen, select (Settings) > [System Software Update].
        • Follow the screens to complete the update.
        • If your Ps4 system does not recognise the update file, check that the folder names and file name are correct. Enter the folder names and file name in single-byte characters using uppercase letters.
